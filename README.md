TweetBlind for TweetDeck
==========================

Chrome extension.
Blind the shitty tweet from your timeline.

## Installation
### Google Chrome

- Download/git clone this project
- Goto chrome://extensions/
- Click `Load unpacked` button
- Select TweetBlind directory
